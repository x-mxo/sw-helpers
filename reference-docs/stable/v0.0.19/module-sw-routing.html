Type Definitions

this.Matcher(input) → Array.<Object>|null

A function that can be automatically invoked by a route to determine whether or not an incoming network request should trigger the route's handler.

Parameters
input Object
Properties
url URL

The request's URL.
event FetchEvent

The event that triggered the fetch handler.
Returns Array.<Object> | null

To signify a match, return a (possibly empty) array of values which will be passed in a params to the RouteHandler. Otherwise, return null if the route shouldn't match.

this.RouteHandler(input) → Promise.<Response>

A handler that can be automatically invoked by a route, and knows how to respond to a request. It can either be a standalone function or a subclass of Handler.

Parameters
input Object
Properties
url URL

The request's URL.
event FetchEvent

The event that triggered the fetch handler.
params Array.<Object>

Any additional parameters that the Route provides, such as named parameters in an ExpressRoute.
Returns Promise.<Response>

The response that will fulfill the request.
