Router

The Router takes one or more Routes and registers a fetch event listener that will respond to network requests if there's a matching route.

It also allows you to define a "default" handler that applies to any requests that don't explicitly match a Route, and a "catch" handler that responds to any requests that throw an exception while being routed.

Example

// The following example sets up two routes, one to match requests with
// "assets" in their URL, and the other for requests with "images", along
// different runtime caching handlers for each.
// Both the routes are registered with the router, and any requests that
// don't match either route will be handled using the default NetworkFirst
// strategy.
const assetRoute = new RegExpRoute({
  regExp: /assets/,
  handler: new goog.runtimeCaching.StaleWhileRevalidate(),
});
const imageRoute = new RegExpRoute({
  regExp: /images/,
  handler: new goog.runtimeCaching.CacheFirst(),
});

const router = new goog.routing.Router();
router.registerRoutes({routes: [assetRoute, imageRoute]});
router.setDefaultHandler({handler: new goog.runtimeCaching.NetworkFirst()});

Constructor

new Router()

Start with an empty array of routes, and set up the fetch handler.

Instance Methods

this._addFetchListener()

This method will actually add the fetch event listener.

this.registerRoute(input)

Registers a single route with the router.

Example
router.registerRoute({
  route: new Route({ ... })
});
Parameters
input Object
Properties
route Route

The route to register.

this.registerRoutes(input)

Register routes will take an array of Routes to register with the router.

Example
router.registerRoutes({
  routes: [
    new RegExpRoute({ ... }),
    new ExpressRoute({ ... }),
    new Route({ ... }),
  ]
});
Parameters
input Object
Properties
routes Array.<Route>

An array of routes to register.

this.setCatchHandler(input)

If a Route throws an error while handling a request, this {RouteHandler} will be called and given a chance to provide a response.

Example
router.setCatchHandler(({event}) => {
  if (event.request.mode === 'navigate') {
    return caches.match('/error-page.html');
  }
  return Response.error();
});
Parameters
input Object
Properties
handler RouteHandler

The handler to use to provide a response.

this.setDefaultHandler(input)

An optional {RouteHandler} that's called by default when no routes explicitly match the incoming request.

If the default is not provided, unmatched requests will go against the network as if there were no service worker present.

Example
router.setDefaultHandler({
  handler: new goog.runtimeCaching.NetworkFirst()
});
Parameters
input Object
Properties
handler RouteHandler

The handler to use to provide a response.
