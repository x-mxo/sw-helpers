Instance Methods

this.escapeRegExp(string) → string

Escaping user input to be treated as a literal string within a regular expression can be accomplished by simple replacement.

From https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_Expressions

Parameters
string string

The string to be used as part of a regular expression
Returns string

A string that is safe to use in a regular expression.
